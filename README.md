### Installation and building

1. Clone repository

```sh
git clone https://github.com/FilenCloudDienste/filen-web filen-web
```

2. Update dependencies

```sh
cd filen-web && npm install
```

3. Running a development build

```sh
npm run dev
```

4. Build

```sh
npm run build
```
